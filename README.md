# SMS-Spam-Detection-Using-Machine-Learning

## 📌 Overview

This project is designed to classify SMS messages as **Spam** or **Ham (Not Spam)** using Machine Learning and Natural Language Processing (NLP) techniques.

The system preprocesses text messages, extracts features using TF-IDF Vectorization, and predicts whether a message is spam or not.

---

## 🎯 Objectives

* Detect unwanted spam messages automatically
* Improve SMS filtering accuracy
* Compare multiple machine learning algorithms
* Build a real-time spam prediction system
